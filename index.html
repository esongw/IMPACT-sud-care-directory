📥 Import Notes

Import Format - CSV File

Upload a CSV file exported from this tool. The following columns are supported:

ID, Name, Secondary Name, Street Address, City, State, County, Hours, General Contacts, Staff Contacts, Patient Notes, Clinical Staff Notes

Multi-entry fields (Hours & Contacts) use this format:
 • Hours: Description|Hours ;; Description|Hours
   Example: Office Hours|Mon–Fri 9am–5pm ;; After Hours|Call 503-555-0000
   (Leave description blank: |Mon–Fri 9am–5pm)
 • Contacts: Type|Info|Notes ;; Type|Info|Notes
   Example: Phone|503-555-1234| ;; Intake|503-555-5678|Call after 9am

Tip: Use the "Export Template" button to get a pre-filled file in the correct format.
